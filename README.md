# Student Performance Predictor (AI & ML Project)

## Overview
This project predicts student academic performance using Machine Learning techniques. It analyzes factors such as study hours, sleep hours, attendance, and previous marks to estimate final exam scores.

## Problem
Students often struggle to understand how their daily habits impact academic performance. This project helps predict outcomes and analyze patterns.

## Features
Student score prediction
Machine learning model training
Data visualization
Accuracy evaluation
User input prediction
